В проекте было реализовано 4 этапа:
1. Минимальный прототип с конфигурацией
2. Сбор данных из пакетов формата Rust (Cargo)
3. Основные операции - реализация алгоритма DFS без рекурсии
4. Дополнительные операции - вывод порядка зависимостей
Для каждого этапа были созданы тестовые команды, а там где требуется - тестовые
репозитории.
В конечном итоге был разработан инструмент визуализации графа зависимостей для
менеджера пакетов, соответсвующий всем описанным в задании требованиям.
